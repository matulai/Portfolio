# My Portfolio

## Tecnologies

- **Next.js**
- **Tailwind**
- **Typescript**

## Getting Started

### Requeriments

- **Node.js** 18.18 or later.

### install all dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
